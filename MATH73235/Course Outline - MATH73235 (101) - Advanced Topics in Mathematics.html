Course Description

This course is divided between advanced topics selected from calculus, statistics and discrete mathematics. Topics are selected to support advanced studies in natural sciences and engineering. Commonly selected topics would include aspects of partial derivatives, surface integrals, spherical and cylindrical coordinates, methods of proof, number and set theory, stochastic processes and queuing theory.

Program Outcomes Supported

Course Outcomes

Successful completion of this course will enable the student to:
  1. Represent and solve problems in cylindrical and spherical coordinate systems
  2. Model physical phenomena using Partial Differential Equations (PDEs), such as string vibration, membrane vibration, wave propagation and heat transfer
  3. Solve PDEs using various methods, including Fourier and Laplace transforms
  4. Perform operations on vectors in 2D and 3D Cartesian and curvilinear spaces
  5. Compute properties of vector functions
  6. Define and compute double and triple integrals over general regions
  7. Compute line and surface integrals
  8. Apply graph theory to network flow problems and assignment problems
  9. Explain and formulate methods of mathematical proofs such as direct proof, proof by contraposition, and proof by contradiction.

Unit Outcomes

Successful completion of the following units will enable the student to:
1.0     Coordinate Systems
1.1     Represent points and vectors in Cartesian, Cylindrical, and Spherical 3D spaces.
1.2     Convert between Cartesian and Cylindrical coordinates
1.3     Convert between Cartesian and Spherical coordinates
2.0     Partial Differential Equations (PDEs)
2.1     Discuss concepts and applications of PDEs
2.2     Model a vibrating string using the Wave Equation
2.3     Solve the wave equation using the Fourier Series
2.4     Solve the wave equation using D’Alembert’s method
2.5     Model a vibrating rectangular membrane using the two-Dimensional Wave Equation
2.6     Solve two-dimensional wave equations using Double Fourier Series
2.7     Model heat flow using the Heat Equation
2.8     Solve the heat equation using the Fourier Series.
2.9     Solve the steady state two-dimensional heat problem
2.10     Solve the heat transfer equation along a bar using Fourier Integrals and Transforms
2.11     Discuss Laplace’s Equation in Cartesian, Cylindrical and Spherical Coordinates
2.12     Use Laplacian in Polar Coordinates to solve a circular membrane problem
2.13     Solve PDEs using the Laplace Transforms
3.0     Vector Calculus
3.1     Discuss the properties, calculation, and applications of the gradient, divergence, and curl.
3.2     Evaluate line and surface integrals
3.3     Discuss the properties of line and surface integrals
3.4     State Green’s Theorem and use it to evaluate path integrals
3.5     State Gauss’s theorem (the Divergence Theorem) and use it to evaluate surface and volume integrals
3.6     State Stoke’s Theorem and use it to evaluate surface integrals
4.0     Graph theory
4.1     Discuss flow problems in networks
4.2     Find flow augmenting paths in a network
4.3     Determine the maximum flow for a network using Ford-Fulkerson algorithm
4.4     Discuss bipartite graphs and assignment problems
5.0     Methods of Proof
5.1     Discuss methods of mathematical proofs such as: direct proof, indirect proof, proof by contradiction, proof by induction, and case analysis
5.2     Formulate and present a rigorous and logical mathematical proof
5.3     Investigate and explain illustrative and well-known mathematical proofs
